# PsPOverclock_V1 

Плагин для разгона PSP  
Не требует никаких настроек, просто кидаете
PsPOverclock_V1.prx в папку seplugins и 
добавляете в файл GAME.TXT строку
  
ef0:/seplugins/PsPOverclock.prx 0  
  
Активируете из меню рекавери и всё, разгон работает,
деактивируете и разгон выключается.
  
На данный момент проверенно на пяти версиях.  

Версия:        Разгон:  
TA-085v1 (02g) ~25%  
TA-088v1 (02g) ~25%  
TA-090v2 (03g) ~25%  
TA-093v1 (04g) ~15%  
TA-091   (06g) ~16%  

На разных версиях разный тактовый генератор,
что и не позволило разогнать одинаково. Ещё
тестироваллось на стритке, и даже работало,
но это не точно. Пока добавить в список не могу.
Во время разработки и тестированния никаких 
проблем связанных с разгоном не наблюдалось.

В данной версии есть проблеммы с выходом из
игры в разгоне и загрузки следующей. Иногда
это приводит к зависанию или снижению частоты.
Лечится перезагрузкой.  

ВНИМАНИЕ: не запускать плагин на PS Vita.
